# Pack Python Project into Docker Image

## Prerequirements
Bash version 4.x+, docker, and internet connection 

  
### Build 
   ```
   cd aicots-template/
   curl https://raw.githubusercontent.com/juouyang-aicots/py2docker/main/build.sh | bash
   ```

### Run
  ```
   cd aicots-template/
   bash run.sh
   ```
